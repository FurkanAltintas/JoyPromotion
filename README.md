# JoyPromotion
A simple demo site made using Dapper and Asp.Net Core

### Project Details
------------
- N Layered Architecture
- Dapper & Dapper.Contrib
- Asp.Net Core
- Asp Net Core Custom Cookie Auth.
- AutoMapper
- Fluent Validation
- Code First
- Dto (Data Transfer Object)
- ViewModel
- Enum
- Html5
- Css3
- JavaScript
- Ajax
- Select2
- SummerNote
